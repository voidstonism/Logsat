# Logsat

This is open source and free code for loggers in the ROBLOX game "Waste of Space [Alpha]"


## [SETUP]

  The following parts can be connected directly to the 'Microcontroller' without ports via cjoints, 'EthernetCable' or direct weld.

  To setup Discord webhooks refer to: https://support.discord.com/hc/en-us/articles/228383668-Intro-to-Webhooks

  ### [RECOMMENDED]
  
    'Modem'
    
    'Micophone'
    
    'Instrument'
    
    'LifeSensor'
    
    'Telescope' or 'Sign' -- To identify your logger or else it will be considered "Unknown"
    
  ### [OPTIONAL]
  
    'BlackBox'
